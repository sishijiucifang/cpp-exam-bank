# C++ 程序设计题库

国防科技大学《大学计算（下）》C++程序设计题库，按章节知识点整理，共 **385 道**编程题。

```
├── 第1章-C++基本语法/     (42 题)  基本语法、数学公式、简单算法
├── 第2章-流程控制/        (47 题)  if-else、for/while 循环、嵌套循环
├── 第3章-函数/            (55 题)  函数定义与调用、引用传参、递归函数
├── 第4章-数组与综合设计/  (37 题)  一维/二维数组、综合算法设计
├── 第5章-指针/            (41 题)  指针声明、指针运算、动态内存分配
├── 第6章-数组与字符串/    (68 题)  数组操作、C风格字符串处理
├── 第7章-结构体与链表/    (12 题)  struct定义、单向链表增删改查
├── 第8章-文件操作/        (13 题)  ifstream/ofstream 文件读写
├── 第9章-综合程序设计/    (16 题)  链表、文件、数组综合应用

├── final-25/              (20 题)  2025-2026学年期末考试 A/B 卷
```

## 题目来源

- **C++题库.txt** — 原始题库，包含试卷编号9564（第1-5章，221题）和9565（第6-13章，119题），共340题
- **final-24/** — 2023-2024学年期末考试真题，A/B卷各10题
- **final-25/** — 2025-2026学年期末考试真题，A/B卷各10题

## 文件命名

| 前缀 | 说明 | 示例 |
|------|------|------|
| `T{题号}_` | 题库题目 | `T636_约瑟夫问题.cpp` |
| `E{年份}{AB卷}Q{题号}_` | 考试题目 | `E24AQ09_约瑟夫问题.cpp` |
| `P{序号}_` | 专项练习 | `P1_problem1-count.cpp` |

## 使用说明

每个 `.cpp` 文件包含：
- 题目标题和描述（注释头部）
- 完整的骨架代码
- 需要补全的代码区域（`/**********Program**********/` 和 `/**********  End  **********/` 之间）
- 参考答案（注释形式保存）

直接在 Visual Studio Code 等 IDE 中打开对应文件即可练习。

## 考试说明

期末考试为闭卷机试，题型为程序填空补全。每道题提供完整的程序框架，需在指定区域内填写代码实现题目要求的功能。
